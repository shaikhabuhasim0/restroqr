# 🍽️ RestroQR — QR Code Based Restaurant Ordering System

A smart dine-in ordering web app where customers can scan a QR code on their table and place orders directly — no waiter needed!

---

## 🚀 Live Demo
🔗 [restroqr-production.up.railway.app](https://restroqr-production.up.railway.app)

---

## ✨ Features
- 📱 QR Code based table ordering for customers
- 🛒 Customer-facing menu & cart UI
- 🔐 Admin panel to manage orders & menu
- ⚡ Real-time order management
- 💾 Lightweight SQLite database

---

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python (Flask) |
| Database | SQLite |
| Frontend | HTML, CSS, JavaScript |
| Deployment | Railway + Gunicorn |

---

## ⚙️ How to Run Locally
```bash
# 1. Clone the repo
git clone https://github.com/shaikhabuhasim0/restroqr

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
cd backend
python app.py
```
App will run on `http://localhost:5000`
